# Vocab Master v7.0 — Feature Guide

## What changed in this version
- **Set Doctor** diagnoses the selected set and opens a repair mission.
- **Memory Coach** adds a layered **Retrieval Ladder**, **Context Replay**, and **Use It Now** prompt inside study modes.
- **Rescue Lane**, **Set Intelligence**, **Smart Suggestions**, and **Memory Constellation** now work together more clearly.

## Fast start
1. Add or import words into the correct set.
2. Open **Set Doctor** before adding more items.
3. Use **Memory Rescue** or the recommended plan to clear weak and due items.
4. In study mode, reveal hints one step at a time through **Memory Coach**.
5. End with one short **Use It Now** sentence.

## Core features and how to use them
### Set Doctor
- Reads the selected set and checks health, context coverage, overlap, and weak clusters.
- Use **Launch doctor mission** when the set still has due or weak items.
- Use **Switch suggestion lane** only after reading the diagnosis.

### Memory Coach
- **Retrieval Ladder** gives hints in layers so recall effort stays active.
- **Context Replay** shows set, entry type, memory stage, example, note, and tags.
- **Use It Now** gives one short production prompt to move the word into active use.

### Rescue Lane
- Appears after mistakes or whenever you open it manually.
- Use it to queue a nearby related word or save a bridge word quickly.

### Memory Constellation
- Builds cluster missions for topic families, confusion zones, and high-risk groups.
- Use it when normal card-by-card review feels repetitive.

### Language lanes
- **English Focus**: more immersion.
- **Balanced Bilingual**: easiest transition mode.
- **Pure Vietnamese**: lowest cognitive load for heavy review sessions.

## Best routine
- Start with **Memory Rescue**.
- Use **Set Doctor** after importing new material.
- Use **Typing** or **Dictation** to repair weak recall.
- Reveal hints before opening Rescue Lane.
- Finish with one short active-use sentence.

## Data safety
- Export JSON backups regularly.
- Reload the same unpacked extension when updating to reduce storage risk.
