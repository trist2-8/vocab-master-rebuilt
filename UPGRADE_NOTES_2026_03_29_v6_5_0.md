# Vocab Master v6.5.0

## Main upgrade
- Rebuilt the learning mode selection UI into a new **Learning Lab** layout.
- Added **Rescue Lane** support suggestions during study.

## Rescue Lane behavior
- When you forget, answer incorrectly, or ask for help, the system now suggests:
  - a related word already in your study pool, or
  - a new comparison / bridge word to save quickly.
- The strongest related word can be moved into the **next study slot** automatically after a mistake.

## UI changes
- New spotlight card for the recommended mode.
- Redesigned mode cards with clearer purpose and use-case.
- Added inline hint buttons in Flashcard, Quiz, Typing, and Dictation.

## Goal of this version
Reduce blind repetition and turn mistakes into guided transitions toward nearby useful words.
