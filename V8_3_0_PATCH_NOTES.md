# V8.3.0 patch notes

## Main upgrades
- Added a new Growth Cockpit to the Review screen with streak, goal progress, weakest skill, session tempo, and one-tap quick lanes.
- Added adaptive session sizes: Sprint, Flow, and Deep.
- Added Context Burst quick launch for example-driven and sentence-driven learning.
- Added idle cache prewarm for review and set analytics to reduce waiting when moving into review-heavy screens.
- Added two new liquid-glass presets: Prism Frost and Ink Glass.
- Polished the glass clarity of the management / studio surfaces.

## Performance direction
- Review, contrast, and set context caches are warmed in the background after startup and after saves.
- Session queues are trimmed to the chosen tempo so study runs feel lighter and more consistent.

## Safety
- This update is additive and keeps the current storage identity intact when installed into the same unpacked extension folder.
