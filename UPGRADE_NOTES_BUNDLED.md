# Vocab Master Bundled Upgrade

Version 4.0.0

This bundled update keeps Vocab Master focused on vocabulary learning and long-term memory fluency while adding a lightweight motivation/customization layer:

- Theme shop with unlockable UI themes paid by cosmetic coins
- Card style unlocks
- UI density and motion toggles
- Animated study companion and celebration effects (optional)
- Built-in Pomodoro timer with bonus cosmetic coins
- Daily mission strip tied to real study activity

Important design choice:
- Core learning logic and memory-first review flow remain unchanged
- Cosmetic spending uses earned study coins plus separate focus bonuses, while tracking spent cosmetic coins independently so the main learning system stays stable
