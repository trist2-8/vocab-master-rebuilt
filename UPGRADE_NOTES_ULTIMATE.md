# Vocab Master — Ultimate Upgrade

This build keeps the original purpose intact: improve vocabulary learning and long-term memory fluency while preserving simplicity and ease of use.

## Included upgrades
- Guided study with smarter recommendations
- Confidence-based memory stages
- Quick-focus study buttons for due / weak / new words
- Typo-tolerant answer checking for typing and dictation
- More flexible meaning matching
- Daily goal, streak, and study momentum tracking
- Dashboard insight cards and recent study summaries
- Search + sort for word sets
- Search + memory-stage filters inside set details
- JSON backup export/import
- Expanded quick-add fields (phonetic, example, notes)
- Session summary modal after each study session
- Category/type kept before meaning in add/edit/import flows

## Core principle
Every upgrade in this build is meant to strengthen learning and retention without adding unnecessary complexity or drifting away from the original foundation.
